Метод – это встроенная команда («функция», мы поговорим о них позже), которую нужно вызвать для получения значения. При обращении без скобок мы получим саму эту функцию. Как правило браузер выведет её как-то так: "function toUpperCase() { ... }". А чтобы получить результат – нужно произвести её вызов, добавив скобки Есть методы и у чисел, например num.toFixed(n). В JavaScript можно использовать букву "e", чтобы укоротить запись числа. Она добавляется к числу и заменяет указанное количество нулей let billion = 1e9; // 1 миллиард, буквально: 1 и 9 нулей alert( 7.3e9 ); // 7.3 миллиардов (7,300,000,000) et ms = 1e-6; // шесть нулей, слева от 1 если дроби,то всегда ставится точка 0.111 не / Обььект .Math содержит в себе огромное количество методов для работы с математикое, это встороенные функции для js В JS и в других языках програмирования есть особенности работы с дробями 0.1+0.2=0.30000000000000004 Не точна работа с цифрами после запятой, если работать с деньгами, можно все хранить в копейках, а потом просто делить на сто вывод результата .Math max min - возвращают максимальное и минимальное число из введенных аргументов, количество аргументов может быть не ограничено Внимание! Две точки в 123456..toString(36) это не опечатка. Если нам надо вызвать метод непосредственно на числе, как toString в примере выше, то нам надо поставить две точки .. после числа. 123456..toString() '123456'
